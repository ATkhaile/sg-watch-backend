# GUIツールでカラム追加する手順

## PHPMyAdmin を使用する場合

1. **テーブルを選択**
   - `work_logs` テーブルを選択

2. **構造タブを開く**
   - 「構造」タブをクリック

3. **カラムを追加**
   - 「category」カラムの後に「追加」をクリック
   - または下部の「カラムを追加」ボタンをクリック

4. **新しいカラムの設定**
   
   **1つ目: role_percentages**
   - カラム名: `role_percentages`
   - 型: `JSON`
   - 長さ/値: (空白)
   - デフォルト: `NULL`
   - NULL: チェック
   - 照合順序: (デフォルト)
   - コメント: `役割別工数配分（DS,QA,PM,PMO,PG,SE）`

   **2つ目: primary_role**
   - カラム名: `primary_role`
   - 型: `ENUM`
   - 長さ/値: `'DS','QA','PM','PMO','PG','SE'`
   - デフォルト: `NULL`
   - NULL: チェック
   - 照合順序: (デフォルト)
   - コメント: `主要役割`

5. **保存**
   - 「保存」ボタンをクリック

## HeidiSQL を使用する場合

1. **テーブルを選択**
   - 左側のツリーで `work_logs` テーブルを選択

2. **テーブル設計者を開く**
   - 右クリック → 「テーブル設計者」を選択

3. **新しい行を追加**
   - 「category」行の下に新しい行を2つ追加

4. **カラム設定**
   - 上記のPHPMyAdminと同じ設定を入力

5. **保存**
   - Ctrl+S または「保存」ボタンをクリック
